# Lua Minecraft Project

This is a Lua-based project for a Minecraft game that I am building together with my siblings. We are using [Rojo](https://github.com/rojo-rbx/rojo) to manage our development workflow and sync code.

## Development Setup

To build the project from scratch, use:

```bash
rojo build -o "minecraft_project.rbxlx"
```

To start the development server and sync your changes in real-time:

```bash
rojo serve
```

For more information on the tools being used, check out [the Rojo documentation](https://rojo.space/docs).
