# Roblox Minecraft Project

This is a **Roblox** project for a Minecraft-style game that I am building together with my siblings using Lua. We are using [Rojo](https://github.com/rojo-rbx/rojo) to manage our development workflow and sync code from VS Code to Roblox Studio.

## Development Setup

To build the project from scratch, use:

```bash
rojo build -o "minecraft_project.rbxlx"
```

To start the development server and sync your changes in real-time:

```bash
rojo serve
```

For more information on the tools being used, check out [the Rojo documentation](https://rojo.space/docs).
